# Gestor de Tareas Full Stack

Proyecto desarrollado con **Angular** (Frontend), **Node.js** (Backend) y **MySQL** (Base de datos).

## Características
* Listado de tareas en tiempo real.
* Separación de tareas pendientes y completadas.
* Contador de tiempo transcurrido desde la creación.
* CRUD completo (Crear, Leer, Actualizar, Eliminar).

## Cómo ejecutar
1. Clonar el repositorio.
2. En `/backend`: Ejecutar `npm install` y `node index.js`.
3. En `/frontend`: Ejecutar `npm install` y `ng serve`.
4. Importar la base de datos `todolist_db`.
